# 冒泡算法排序
> 冒泡算法是一种交换排序,每一个将最大/最小的数字 移动到末尾
## 思考:
1. 假设有n个数,每次遍历将最大/最小的数字交换到最后.
2. 对剩下的n-1个数重复1的步骤.
3. 当只剩下一个数字,或者某次遍历过程未发生交换时,排序结束.

## 代码:
**bubbleSort.c**
```c
void bubbleSort(int keys[], int n, int asc)
{
    int exchange = 1;
    for (int i = 1; i < n && exchange; i++)
    {
        exchange = 0;
        for (int j = 0; j < n - i; j++)
        {
            if (asc ? keys[j] > keys[j + 1] : keys[j] < keys[j + 1])
            {
                swap(keys, j, j + 1);
                exchange = 1;
            }
        }
    }
}
```
